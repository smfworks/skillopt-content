# Optional profiles

Core `skillopt-content` is **generic on purpose**.

Put organization-specific material here (or in a private repo):

- House voice and tone rules  
- Privacy / redaction policies  
- Publishing CTAs and channel conventions  
- Brand handles and disclosure language  

## How to use

1. Create `profiles/<your-org>.md` **locally or in a private fork**.  
2. Load it **after** `skill_template.md` in your agent harness.  
3. Do **not** submit private profiles as the default skill in PRs to this repo.
